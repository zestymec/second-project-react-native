# React Native UI Practice Components 📱

This repository contains a collection of essential UI components built while learning React Native. The project focuses on mastering layouts, styling, and functional logic using TypeScript.

## 🚀 Features
- **Flat & Elevated Cards**: Mastered Flexbox and horizontal `ScrollView`.
- **Fancy Image Cards**: Styled local and network images with rounded layouts.
- **Dynamic Contact List**: Used `.map()` to render data from objects.
- **Action Cards**: Integrated `Linking API` to open external URLs (YouTube/Web).

## 🛠️ Tech Stack
- **Framework**: React Native (CLI)
- **Language**: TypeScript / JavaScript (ES6)
- **Library**: React Hooks
